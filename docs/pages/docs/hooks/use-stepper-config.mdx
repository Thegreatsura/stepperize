# useStepperConfig Hook

The `useStepperConfig` hook provides access to the stepper configuration within a `Stepper`. It allows you to interact with the stepper's configuration props.

We refer to values such as `orientation`, `expandable`, `searchParams`, etc.

## Usage

Here is an example of how to use the `useStepperConfig` hook:

```tsx
import { Stepper, StepperStep, createSteps, useStepper, useStepperConfig } from "@stepperize/react";

const steps = createSteps([
  { id: "step-1", title: "Step 1", description: "Description for step 1" },
  { id: "step-2", title: "Step 2", description: "Description for step 2" },
  { id: "step-3", title: "Step 3", description: "Description for step 3" },
] as const);

const MyStepperComponent = () => {
  return (
    <Stepper steps={steps}>
      <StepperContent />
    </Stepper>
  );
};

const StepperContent = () => {
  const { steps } = useStepper();
  const { orientation } = useStepperConfig();

  if(orientation === "horizontal") {
    return null
  }

  return (
    <div>
      {steps.map((step, index) => (
        <StepperStep key={step.id} index={index}>
          {/* Step content */}
        </StepperStep>
      ))}
    </div>
  );
};
```
