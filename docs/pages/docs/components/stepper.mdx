# Stepper Component

The `Stepper` component is a versatile and customizable component that allows you to create step-based workflows in your React application. It supports both horizontal and vertical orientations, and can be configured to be expandable, clickable, display various icons for different states, etc.

Internally it works with the `Context` API so we can use the `useStepper` and `useStepperConfig` hooks in any child component.

## Usage

Here is an example of how to use the `Stepper` component:

```tsx
import { Stepper, createSteps } from '@stepperize/react';

const steps = createSteps([
  { id: 'step-1', title: 'Step 1', description: 'Description for step 1' },
  { id: 'step-2', title: 'Step 2', description: 'Description for step 2' },
  { id: 'step-3', title: 'Step 3', description: 'Description for step 3' },
] as const);

const MyStepperComponent = () => {
  return (
    <Stepper steps={steps}>
      {/* Stepper content */}
    </Stepper>
  );
};
```
