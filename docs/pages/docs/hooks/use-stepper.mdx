# useStepper Hook

The `useStepper` hook provides access to the stepper actions and state within a `Stepper`. It allows you to interact with the stepper's steps, handle keyboard navigation, and manage step states.

## Usage

Let's look at an example where we create the step actions using the `useStepper` hook.

```tsx
import * as React from "react";

import { useStepper } from "@stepperize/react";
import { Button } from "@components/ui/button";

export const StepperActions = () => {
  const {
    isLastStep,
    isOptionalStep,
    goToNextStep,
    goToPrevStep,
    canGoToNextStep,
    canGoToPrevStep,
    reset,
    isStepperFinished,
  } = useStepper();

  return (
    <React.Fragment>
      {isStepperFinished && (
        <div className="h-40 w-full flex items-center justify-center border bg-secondary text-primary rounded-md">
          <h1 className="text-xl">Woohoo! All steps completed! ðŸŽ‰</h1>
        </div>
      )}
      <div className="w-full flex justify-end gap-2">
        {isStepperFinished ? (
          <Button type="button" onClick={reset}>
            Reset
          </Button>
        ) : (
          <>
            <Button
              variant="ghost"
              disabled={!canGoToPrevStep}
              onClick={goToPrevStep}
            >
              Prev
            </Button>
            <Button disabled={!canGoToNextStep} onClick={goToNextStep}>
              {isLastStep ? "Finish" : isOptionalStep ? "Skip" : "Next"}
            </Button>
          </>
        )}
      </div>
    </React.Fragment>
  );
};
```

If you need the hook to return the steps with inferred values, you can do this:

```tsx {7,9}
const steps = createSteps([
  { id: "step-1", title: "Step 1", description: "Description for step 1" },
  { id: "step-2", title: "Step 2", description: "Description for step 2" },
  { id: "step-3", title: "Step 3", description: "Description for step 3" },
] as const);

type Steps = typeof steps;

const { steps, ...rest } = useStepper<Steps>(); 
```

In the case of useStepper, the `steps` array is of type `ExtendedStep[]` which extends `Step` with attributes `data-` and `aria-`. 
These objects are very useful to get free keyboard navigation and accessibility and to implement them in buttons or anchor tags.

For example:

```tsx
const StepperContent = () => {
	const { steps } = useStepper<Steps>();

	const lastStepIndex = steps.length - 1;

	return (
    <nav
      role="tablist"
      aria-orientation="vertical"
      className="flex flex-col gap-4"
    >
      <ol className="w-full flex flex-col">
        {steps.map((step, index) => (
          <div key={step.id} {...step?.dataAttr} className="group">
            <li className="flex gap-2 items-center">
              <a {...step?.triggerProps} className={classForStepTrigger()}>
                <span>Step {index + 1}</span>
              </a>
              <span>{step.title}</span>
            </li>
            <div className="flex group-data-[last=false]:my-2 group-data-[last=true]:group-data-[completed=false]:mt-4">
              {index !== lastStepIndex && (
                <div className="ms-5 min-h-4">
                  <hr className="border border-border group-data-[completed=true]:border-primary h-full" />
                </div>
              )}
              <StepperStep key={step.id} index={index}>
                <h1 className="text-xl">{step.title}</h1>
              </StepperStep>
            </div>
          </div>
        ))}
      </ol>
    </nav>
	);
};
```
