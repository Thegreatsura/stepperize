# StepperStep Component

The `StepperStep` component is used to render the content of a step within the `Stepper` component. It ensures that only the current active step's content is displayed, and it respects the `expandable` and `orientation` configurations of the `Stepper`.

Internally, it uses the `useStepper` and `useStepperConfig` hooks to determine the current state and configuration of the stepper.

## Usage

Here is an example of how to use the `StepperStep` component:

```tsx
import { Stepper, StepperStep, createSteps } from "@stepperize/react";

const steps = createSteps([
  { id: "step-1", title: "Step 1", description: "Description for step 1" },
  { id: "step-2", title: "Step 2", description: "Description for step 2" },
  { id: "step-3", title: "Step 3", description: "Description for step 3" },
] as const);

const MyStepperComponent = () => {
  return (
    <Stepper steps={steps}>
      {steps.map((step, index) => (
        <StepperStep key={step.id} index={index}>
          {/* Step content */}
        </StepperStep>
      ))}
    </Stepper>
  );
};
```
