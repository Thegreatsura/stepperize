## What is shadcn-stepper?

Is an abstraction of the [@stepperize/react](https://stepperize.vercel.app/docs/react) library to allow the shadcn/ui community to use the stepper component in their projects.

## Why shadcn-stepper?

Because shadcn/ui needs a component that allows us to create flows of steps in a simple way and with a high level of customisation along with type safety
